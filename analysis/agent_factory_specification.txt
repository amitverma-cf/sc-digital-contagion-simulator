================================================================================
AGENT FACTORY SPECIFICATION
================================================================================

CONFIGURATION
--------------------------------------------------------------------------------
Cohort Size: 100 agents
Random Seed: 42
Personas: 5

SAMPLING STRATEGY
--------------------------------------------------------------------------------
1. Persona Selection: Multinomial sampling based on original distribution
2. Attribute Sampling: Normal distribution per persona with truncation
3. Derived Metrics:
   - Base Stress: Scaled from behavior class (0-100)
   - Resilience: Uniform [0.7, 1.3]
   - Susceptibility: Uniform [0.5, 1.5]
   - Variability: Uniform [0.8, 1.2]

AGENT ATTRIBUTES
--------------------------------------------------------------------------------
Core Attributes:
  • agent_id: Unique identifier (0 to n-1)
  • persona: Behavioral archetype
  • base_stress: Initial stress level (0-100)
  • resilience: Stress resistance multiplier
  • susceptibility: Peer influence multiplier
  • variability: Daily usage variance

Behavioral Attributes (from persona):
  • app_usage: Daily app usage (minutes)
  • screen_time: Daily screen time (hours)
  • battery_drain: Daily battery drain (mAh)
  • apps_installed: Number of apps
  • data_usage: Daily data consumption (MB)
  • behavior_class: User behavior category (1-5)
  • age: Agent age (18-59)

VALIDATION CRITERIA
--------------------------------------------------------------------------------
Synthetic cohort aggregate statistics must match original dataset:
  • Mean differences < 10% for key metrics
  • Persona distribution within ±5% of original
